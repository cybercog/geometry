Geo I/O Geometry
================

[![Build Status](https://secure.travis-ci.org/geo-io/geometry.png?branch=master)](http://travis-ci.org/geo-io/geometry)

Basic geometric objects.

Installation
------------

Install [through composer](http://getcomposer.org).

```json
{
    "require": {
        "geo-io/geometry": "0.1.*@dev"
    }
}
```
